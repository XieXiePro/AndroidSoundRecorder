# 一款用于在Android设备上获取（录音或从系统文件中选择）音频的开源工具库

### 使用说明

    通过 SoundRecorder.launchAct(Activity activity)启动录音程序。
    通过 SeleteAudioActivity.launchAct(Activity activity)从系统文件选择音频。

### 版本说明

    V1.0.0
        * 支持录音 *
        * 支持从系统选择音频 *
        * 实现录音转化为MP3音频格式 *
        * 实现录音重命名并保存 *
        * 改写自Android原生录音器 *
